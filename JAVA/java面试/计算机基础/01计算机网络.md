## 传输层（运输层）

传输层提供了进程间的逻辑通信，相当于两个软件之间的通信，如用qq给你的qq发消息

TCP三次握手：

1. 客户端：打电话
2. 服务器端：接电话，说喂你好
3. 客户端：说你好

进行三次握手的原因是：因为客户端的连接请求在网络中有延时，客户端等待一会后会再发送一次请求。如果不进行三次握手服务器会打开两个连接，如果有三次握手，则客户端只会响应第一个服务器的连接请求，会忽略后面的连接请求，则只会有一个请求完成三次连接，建立连接。
