# 1、函数

```python
def 函数名([输入参数]):
  函数体
  [return xxx]

```

### 1.参数传递两种方式

位置实参`sum(2,3)`

关键字实参`sum(a=3,b=2)`

### 2.在函数调用的时候参数的变化

如果是不可变对象（如：字符串，元组），传递参数不会影响实参的值

如果是可变对象（如：列表，字典，集合），传递参数会影响实参的值

### 3.函数的返回值

如果函数没有返回值，return可以省略

如果函数有一个返回值，返回值类型为原类型（是什么类型就是什么类型）

如果函数有多个返回值，返回值类型为元组

### 4.函数调用的默认参数

函数定义时，给形参设置默认值，只有给默认值不符合的时候才需要设置实参

```python
def sum(a,b=100):
  c=a+b
  return c
print(sum(10))
print(sum(10,20))
print(sum(b=10,a=20))
```

### 5.函数定义时不能确定参数个数的情况

定义个数可变的**位置参数**

```python
def sum(*args):
	print(args)
sum(1)
sum(1,2,3)
#输出
(1,)
(1, 2, 3)
```

定义个数可变的**关键字参数**

```python
def sum1(**args):
	print(args)
sum1(a=1)
sum1(a=1,b=2)
"""输出
{'a': 1}
{'a': 1, 'b': 2}
"""
```

如果是同时有两种参数，则位置形参要放在关键字形参前面

```python
def sum3(*args1,**args2):
	print(args1,args2)
sum3(1,2,a=3)
"""输出
(1, 2) {'a': 3}
"""
```
